# Siglas

| Sigla | Significado         | Descrição                                                 |
| ----- | ------------------- | ----------------------------------------------------------- |
| CR    | Compilation error   | O código foi submetido com erro de compilação            |
| RE    | Runtime error       | O código tenta acessar uma área da memória inválida     |
| TLE   | Time limit exceeded | O código levou mais tempo do que o permitido pelo testador |
| PR    | Presentation error  | A saída apresenta algum erro na formatação esperada      |
| WA    | Wrong answer        | A saída fornece alguma resposta incorreta                  |
